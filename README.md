# Sales Analytics Dashboard (2020–2022)

## Project Overview
This project focuses on building an interactive Sales Analytics Dashboard to analyze revenue, cost, profit, and order trends across products, brands, regions, and supervisors.

The dashboard supports business decision-making through KPI tracking and performance analysis.

---

## Objectives
- Analyze product-wise sales performance
- Track regional sales distribution using map visualization
- Evaluate brand contribution to overall revenue
- Monitor supervisor-level sales performance
- Measure key sales KPIs

---

## Key Performance Indicators (KPIs)
- Total Revenue
- Total Cost
- Total Profit
- Units Sold
- Total Orders
- Total Transactions

---

## Dashboard Features
- Interactive filters for supervisor selection
- Product-wise sales comparison
- Region-wise sales analysis using geographic maps
- Brand-wise contribution analysis
- KPI-based performance monitoring
- Clean and business-focused dashboard design

---

## Tools & Technologies
- Microsoft Excel
- Data analysis and data visualization
- Pivot Tables and Pivot Charts
- KPI calculation using Excel formulas
- Dashboard design and reporting
